# سیستم نوبت دهی مراکز درمانی استان

این سیستم با زبان برنامه نویسی پایتون و فلسک توسعه داده شده است


# مراحل نصب
**دستورات زیر را در مسیر اصلی پروژه درون خط فرمان اجرا کنید**
 - pip install -r requirement.txt
 - py setup.py
 - py run.py

    برای دسترسی به سایت ، پورت 80 لوکال هاست را درون مرورگر خود وارد کنید .

    http://127.0.0.1:80
